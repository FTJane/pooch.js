The main components for the pooch.js javascript library are:

pooch.chart()
pooch.data()
pooch.fetch()
pooch.helpers
pooch.map()
pooch.popup()
pooch.symbols()
pooch.zoomControl()


When a pet project becomes a reality... pooch.js

Create an interactive chart or map in just a few lines:

var chart1 = pooch.chart (".container").symbols ([symbols1, symbols2])
                                       .height (400)
                                       .width (800)
                                       .bounds (symbols1)
                                       .zoomControl (zoomControl1);
